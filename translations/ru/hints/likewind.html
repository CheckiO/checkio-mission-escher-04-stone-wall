Для начала вы можете убрать лишние элементы (0) и оставить непосредственно элементы здания, разделив их построчно. В этом вам помогут такие методы для работы со строками, как replace() и split().
Как мне это описать с помощью Python?
Код будет выглядеть следующим образом: 
plan = plan.replace('0', '')
plan = plan.split('\n')
Первая строка заменяет все '0' на пустую строку (''), вторая - разделяет многострочную строку на отдельные строки, используя в качестве разделительного элемента символ переноса строк - '\n'.
Как мне посчитать площадь, необходимую для строительства дома?
Чтобы определить площадь необходимого прямоугольного участка земли, вам нужно умножить самый длинный элемент списка plan (фактически - длину дома) на количество таких элементов в списке (фактически - ширину дома). Чтобы определить самый длинный элемент - используйте функцию max(), а для определения длины списка - функцию len().
