Вы можете повернуть массив на 90 градусов по часовой стрелке и проанализировать каждую строку на количество '#' в ней. Индекс строки с минимальным количеством '#' и будет ответом.
Как мне повернуть массив на 90 градусов?
Попробуйте использовать следующий код: 
wall = wall.split('\n')

while '' in wall:
    wall.remove('')

rot_wall = ['' for i in range(len(wall[0]))]

for i in range(len(wall)):
    for j in range(len(wall[0])):
        rot_wall[j] += wall[i][j]
Также, в качестве альтернативного варианта, вы можете перевести исходную многострочную строку в двумерный массив, а затем использовать библиотеку numpy для поворота на 90 градусов.
Как мне определить строку с минимальным количеством '#'?
В данной ситуации вам стоит использовать функцию min(), в сочетании с методом count(), который позволяет узнать количество определенного символа/подстроки в строке.
Мне нужна помощь с реализацией этого на Python.
Данный код вернет строку с наименьшим количеством '#': 
answer = min(rot_wall, key=lambda x:x.count('#'))
После чего вам останется лишь найти индекс этой строки в списке rot_wall.
Как мне найти индекс строки в списке?
Вы можете использовать метод index(), описанный в документации по такой структуре данных, как list.
