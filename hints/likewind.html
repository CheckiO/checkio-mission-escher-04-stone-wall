You can rotate an array 90 degrees in a clockwise direction and analyze each string by the number of '#' in it. The index of the string with the minimum number of '#' will be the answer.
How do I rotate an array 90 degrees?
Try using the following code: 
wall = wall.split('\n')

while '' in wall:
    wall.remove('')

rot_wall = ['' for i in range(len(wall[0]))]

for i in range(len(wall)):
    for j in range(len(wall[0])):
        rot_wall[j] += wall[i][j]
Also, as an alternative, you can translate the input multiline string into a two-dimensional array, and then use the numpy library to rotate 90 degrees.
How do I find a string with a minimum number of '#'?
In this case you should use the min() function, in combination with the count() method, which allows you to find the number of a certain character/substring in a string.

answer = min(rot_wall, key=lambda x:x.count('#'))
